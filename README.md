# CAPITAL1 — Website-Entwürfe

Eye-Catcher-Website-Entwürfe für **CAPITAL1 — Investment & Treuhand AG**, einer in der Schweiz domizilierten Investment- & Treuhand-AG, die Privatanlegern ab 10.000 € Zugang zum Schweizer Immobilienmarkt eröffnet.

Alle Entwürfe basieren auf dem hauseigenen CI-Leitfaden (Farben, Typografie, Tonalität, Sektion-Logik).

## CI-Eckdaten

- **Farben:** Schwarz `#000000`, Capital Blau `#83CFEC` (Akzent), Blau Dunkel `#1F8FBF` (Links/CTAs), Anthrazit `#1A1A1A`, Nebel `#F5F8FA`
- **Typografie:** Helvetica Neue (Web-Stack), große dünne Headlines, viel Weißraum
- **Sektion-Logik:** Hero → Vorteile → Funktionsweise → Kennzahlen → Vertrauen/Zertifizierung → Kontakt

## Technik

Eigenständige HTML-Dateien — kein Build nötig. Einfach im Browser öffnen.
Scroll-Effekte respektieren `prefers-reduced-motion`. Bild-/Video-Assets werden frei lizenziert (Unsplash / Wikimedia Commons) per Link eingebunden; für die Produktion sind lizenzierte oder eigene Medien vorgesehen.
